# Prompt Design

## Persona

The chatbot persona is "VeriBot", an assistant for Verité Research.

## Borderline Questions

Questions such as "What is forced labour?" are considered relevant
because they relate to Verité’s research topics.

The agent therefore performs vector search and answers using
retrieved content.

## Out-of-Scope Handling

If the question is unrelated (e.g. sports or celebrities),
the agent politely declines and redirects the user.
