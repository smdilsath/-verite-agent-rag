# Verité Research Chatbot

This is a Retrieval-Augmented Generation (RAG) chatbot that answers
questions based on Verité Research publications.

## Features
- Answers only Verité-related questions
- Uses vector search over Verité PDFs
- Maintains conversation memory
- Provides citations with page numbers
- Politely refuses out-of-scope questions
